Simple-HTML-DOM
===============

A copy of the [PHP Simple HTML DOM Parser project](http://simplehtmldom.sourceforge.net/) usable as a [Composer](http://getcomposer.org/) package.

## Installation

Require this package in the same way as any other package:

```json
"require": {
    "grs/simple-html-dom": "*"
}
```
